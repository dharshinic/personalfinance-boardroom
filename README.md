# Personal Finance Boardroom

A multi-agent personal finance advisor built on [Neuro SAN](https://github.com/cognizant-ai-lab/neuro-san-studio).
See `architecture.md` for system design and `summary.md` for a project overview.

## Setup

### Prerequisites
- Python 3.10+ (a working neuro-san-studio installation/venv)
- A Google Gemini API key (free tier): https://aistudio.google.com/apikey

### Installation
```bash
git clone https://github.com/dharshinic/personalfinance-boardroom.git
cd personalfinance-boardroom
pip install -r requirements.txt
```
